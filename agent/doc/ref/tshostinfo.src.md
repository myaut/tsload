## tshostinfo

##### Name
tshostinfo - prints information about host in human-readable format.

##### Synopsis
`$ tshostinfo [-x] [-l] [{host|sched|cpu|disk|fs|net|all|vm}] [...]`

##### Description
Options:
   * `-x` - print additional data
   * `-l` - print legend

`$ tshostinfo -j` 
Prints host resources in JSON format
